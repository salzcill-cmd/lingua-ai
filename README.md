## 🎯 Tentang Proyek

**LinguaAI** adalah platform terjemahan real-time yang menghancurkan batasan bahasa menggunakan terjemahan mesin neural. Dibangun sebagai _single-page application_ tanpa dependensi backend, aplikasi ini memberikan pengalaman terjemahan yang mulus dengan antarmuka _dark-mode_ modern yang menyaingi produk SaaS premium.

Proyek ini mendemonstrasikan arsitektur frontend tingkat produksi dengan organisasi kode yang bersih, optimasi performa, pertimbangan aksesibilitas, dan pengalaman pengguna yang poles.

## 🔧 Konfigurasi API

### API Aktif

| Parameter | Nilai |
|---|---|
| **Base URL** | `https://translate.fedilab.app` |
| **Endpoint** | `POST /translate` |
| **Autentikasi** | Tidak diperlukan (gratis) |
| **Rate Limit** | Cukup untuk penggunaan personal |
| **Status** | ✅ Aktif & terverifikasi |

### Ganti API Endpoint

Edit objek `CONFIG` di `script.js` baris ~17:

```javascript
const CONFIG = {
  API: {
    BASE_URL: 'https://translate.fedilab.app',  // ← Ganti di sini
    ENDPOINT: '/translate',
    TIMEOUT: 15000,          // Timeout request (ms)
    MAX_RETRIES: 2,          // Maksimal retry
    RETRY_DELAY: 1000,       // Delay antar retry (ms)
  },
  // ...
};
```

### Self-Host LibreTranslate (Production)

Untuk penggunaan production, disarankan _self-host_ LibreTranslate:

```bash
# Docker (paling mudah)
docker run -ti --rm -p 5000:5000 libretranslate/libretranslate

# Atau install via pip
pip install libretranslate
libretranslate

# Kemudian update BASE_URL ke http://localhost:5000
```
