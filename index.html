FAQ

Pertanyaan yang Sering Diajukan

Semua yang perlu Anda ketahui tentang LinguaAI.

Ya, LinguaAI sepenuhnya gratis. Kami menggunakan API LibreTranslate open-source yang menyediakan terjemahan tanpa batas secara gratis. Tanpa registrasi, tanpa kartu kredit, tanpa biaya tersembunyi.

Kami mendukung lebih dari 100 bahasa termasuk Inggris, Indonesia, Spanyol, Prancis, Jerman, Jepang, Korea, Mandarin, Arab, Hindi, dan banyak lagi. Bahasa baru ditambahkan secara berkala.

Tentu saja. Riwayat terjemahan Anda hanya disimpan di local storage browser. Kami tidak memiliki server yang menyimpan data Anda, tanpa pelacakan, tanpa analitik. Terjemahan Anda tetap di perangkat Anda.

Anda dapat menerjemahkan hingga 5.000 karakter sekaligus. Untuk teks yang lebih panjang, kami sarankan membaginya menjadi bagian yang lebih kecil. API menangani sebagian besar kasus penggunaan dengan efisien.

Koneksi internet diperlukan untuk terjemahan karena kami menggunakan API berbasis cloud. Namun, riwayat terjemahan dan favorit Anda tetap dapat diakses secara offline karena disimpan secara lokal.

Terjemahan mesin neural kami mencapai akurasi tinggi untuk pasangan bahasa umum. Meskipun tidak sempurna untuk konten khusus atau teknis, terjemahan ini dapat diandalkan untuk komunikasi sehari-hari, dokumen, dan keperluan umum.
